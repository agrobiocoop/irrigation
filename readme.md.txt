# 🌱 Avocado Irrigation Calculator

Απλή εφαρμογή **Streamlit** για υπολογισμό αναγκών άρδευσης αβοκάντο με βάση:
- Ηλικία δέντρου
- Τύπο εδάφους
- Καιρικά δεδομένα (Open-Meteo για Βατόλακο Χανίων)

## 🚀 Εκτέλεση τοπικά

```bash
pip install -r requirements.txt
streamlit run irrigation_calculator.py
